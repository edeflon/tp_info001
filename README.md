## TP2

Ligne 200 : modifier le **path** par le votre pour charger vos images
Usage : ./main_t2 <nom-fichier-image>
Touches :
- 'a' : filtre moyenneur
- 'm' : filtre médian
- 's' : réhausement contraste
- 'x' : Sobel X
- 'y' : Sobel Y
- 'g' : filtre gradient
- 't' : filtre seuil
- 'e' : filtre esquisse (incomplet)
